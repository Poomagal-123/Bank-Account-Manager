##Bank Account Manager (Core Java)
A simple console-based Bank Account Management System built using **Core Java** and **HashMap**

##Features
- Create new customer accounts
- Deposit and withdraw money
- Check account balance
- View all customer accounts
- Data managed using HashMap

##Technologies Used
- Java SE (Core Java)
- VS Code
- HashMap Collection Framework

##How to Run

1. Clone or download this repository
2. Open folder in **VS Code**
3. Run `Main.java` (green play button)
4. Follow menu options in the terminal
